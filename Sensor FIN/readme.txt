DESCRIPTION
===========
 The present firmware is used to program the finger(s) of HANDi glove device.  

COMPATIBILITY
=============

 The present firmware is comptible with STM32F10x family microcontrollers.

CONFIGURATION
=============
 No hardware congfiguration is needed. To program the fingers comment\uncomment 
 the proper FINGER definition in DeviceSettings.h and the definition of Q[4][4] 
 in EKF_Lib.c
